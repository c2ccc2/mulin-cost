## 平台介绍

沐霖工时系统(原无鱼工时系统），是一款轻量级工时记录和管理工具，包括项目管理，工时上报，工时日报，工时统计等功能。

沐霖工时系统可通过员工工时上报的方式，来记录项目所花费的工时，帮助企业进行项目工时统计、核算人工成本。实时、动态、真实的展示项目投入。

## 软件架构

Go 1.21+ + MySQL 8.0 + Redis 7.0 + Vue 3.5

本项目采用前后端分离架构，支持多种部署方式：

**后端技术栈：**
- 🚀 **Gin 1.10** - 高性能 HTTP Web 框架
- 💾 **GORM 1.25** - Go 语言 ORM 库
- 🔐 **JWT** - 身份认证机制
- 📦 **Viper** - 配置文件管理
- 📝 **Zap** - 结构化日志记录

**前端技术栈：**
- 🎨 **Vue 3.5** - 渐进式 JavaScript 框架
- ⚡ **Vite 6.3** - 新一代前端构建工具
- 🎯 **Element Plus** - Vue 3 组件库
- 📊 **ECharts 5.6** - 数据可视化图表库
- 🛣️ **Vue Router 4** - 官方路由管理器

**数据库：**
- 🗄️ **MySQL 8.0** - 关系型数据库
- 💨 **Redis 7.0** - 缓存数据库

**部署方式：**
- 🌐 **独立部署** - 前后端分离部署（Nginx + Go 二进制）
- 🐳 **Docker Compose** - 容器化一键部署
- 📦 **单程序嵌入式部署** - Go 二进制内嵌前端资源

## 功能列表

**通知系统（v4.1.0 新增）**

1、通知中心：

- 站内消息通知，实时推送系统消息
- 10 种通知类型：阶段分配、阶段验收、阶段驳回、待验收、工时审核等
- 支持标记已读、批量标记、删除通知、按类型筛选
- 顶部导航栏通知铃铛，实时显示未读数量

2、多渠道推送：

- 支持邮件、企业微信、钉钉、飞书等多渠道通知
- 系统级全局配置 + 个人级接收设置
- 异步发送机制，不阻塞主流程
- 默认关闭，需手动配置并启用

3、工时提醒：

- 定时提醒员工填写工时
- 灵活配置提醒时间、提醒日期（工作日/自定义）
- 精准推送：仅提醒当天未填写工时的员工
- 支持启用/禁用控制

**工时模块**

1、我的工时：

普通工时填报

任务工时填报

工时类型分区块展示

Excel 批量导入工时（v4.1.0 新增）

2、批量导入功能：

- 支持通过 Excel 文件批量导入工时记录
- 智能解析：支持标准日期、Excel 序列号日期等多种格式
- 数据验证：自动验证项目 ID、日期、工时等必填字段
- 错误统计：导入完成后显示成功、失败数量和详细错误信息
- 按日期分组：自动将同一天的多条记录合并处理
- 逐日校验：重复填报检测、工时上限校验

**项目模块**

1、项目管理：项目列表、项目概要、项目工时记录、项目成员管理、项目状态管理。

**任务模块**

1、 项目任务列表：项目任务列表、任务详情、任务进度、任务工时、任务状态管理。

2、 任务日报：项目任务日报列表、任务日报详情。

**审核模块**

1、项目工时审核：项目工时审核列表、工时审核、审核详情。

2、双审核接口支持：项目工时和普通工时并行审核。

**数据统计分析（v4.2.0 新增）**

1、工时统计概览：项目总数、总工时、总任务数、总参与人数 4 个统计卡片，快速掌握全局。

2、计划 vs 实际对比：任务预估工时与实际工时对比图表，支持偏差率计算，评估估算准确性。

3、项目工时核算：项目级工时统计视图，展示总工时、任务数、参与人数、平均工时。

4、项目工时对比图表：柱状图可视化，支持前 20 个项目的工时与任务数量对比。

**数据统计**

1、数据概览统计：人员、项目、总工时、项目工时、普通工时、部门工时统计、明细查询。

2、人员统计：人员工时统计列表。

3、项目统计：项目工时统计列表。

4、填报记录查询：填报记录列表、填报记录详情。

**管理模块**

1、项目管理：创建项目、添加人员、状态管理、删除项目。

2、组织管理：用户管理、角色管理、重置密码、批量删除。

3、职位管理：包括筛选、新增、修改、删除、批量删除等功能。

**部署方式**

1、独立部署：传统前后端分离部署方式。

2、Docker Compose 一键部署：简化部署流程。

3、单程序嵌入式部署：Go 二进制 + 嵌入式前端资源。
