## version
工时系统3.2.1

## 1. 准备docker compose包


### 手动整合
找到 compose-template  和版本发布文件

1. 将后端jar 放到 前者conf/app/目录下

2. 将sql文件 放到前者conf/init/ 目录下


3. 将前端文件全部复制到前者conf/html 目录下

注意不要带目录

### 下载整合包
https://www.wuyusoft.com/download/


## 2 .准备工作

```bash
# 进入目录
cd install/compose



# 设置程序运行权限

chmod +x conf/app/*

```


## 3. 启动服务

```bash
# 方式1.直接启动

docker compose up

# 方式2.后台启动

docker compose up -d

```

浏览器访问 http://localhost
